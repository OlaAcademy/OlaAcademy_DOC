# 视频信息更新
 相关信息 | 内容
 ------ | ------
 访问类型 | POST
 访问地址 | /cour/updVideoInfo
 说明 | 每调用一次，视频点击数+1，同时更新时长总计

###请求参数

 参数名称 | 是否必填 | 类型 | 说明 | 示例数据
 ------ | ------ | ------ | ------ | ------ | ------
 videoId | 是 | 字符串 | 视频id
 timeSpan | 否 | 字符串 | 时长，秒
 
###返回结果示例

```javascript
{
  "message": "成功",
  "apicode": 10000
}

```
